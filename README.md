# mew 0.2 - Hex code editor

> [!CAUTION]
> The current version of this program is unstable, has bugs and is not ready for use.

### Installation
```shell
git chone https://github.com/Ad4ndi/mew
cd mew
cargo build
```

### Usage
```shell
mew [file]
```
